## 🌟 **Quick Deploy Steps:**

### 1. **Create GitHub Repository**
```bash
# Go to github.com → New repository
# Repository name: divya-sumit-engagement
# Description: Beautiful engagement website for Divya and Sumit
# Make it PUBLIC (important!)
```

### 2. **Initialize Git & Push**
```bash
# Navigate to your project folder
cd C:\Users\hppc7\CascadeProjects\invitation-gallery-v2

# Initialize Git
git init
git add .
git commit -m "💕 Divya and Sumit Engagement Website"

# Add Remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/divya-sumit-engagement.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### 3. **Enable GitHub Pages**
1. Go to your repository on GitHub
2. Click **Settings** tab
3. Scroll to **Pages** section
4. Under "Build and deployment":
   - Source: **Deploy from a branch**
   - Branch: **main**
5. Click **Save**

### 4. **Visit Your Live Site!**
🎉 **URL:** `https://YOUR_USERNAME.github.io/divya-sumit-engagement/`

## 🆘 **Troubleshooting:**

### **Common Issues:**
**404 Errors:**
- Check repository is PUBLIC
- Verify branch name (main)
- Check file paths (use relative paths)

**Styles Not Loading:**
- Clear browser cache
- Check CSS file path
- Verify font URLs

**Images Not Showing:**
- Ensure images are in `assets/` folder
- Check file names (case-sensitive)
- Use relative paths: `./assets/image.jpg`

### **GitHub Pages Build Logs:**
1. Go to repository → Settings → Pages
2. Check "Build deployment" section
3. View error messages if any
