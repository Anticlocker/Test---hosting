# 💕 Divya and Sumit Engagement Website

A beautiful, interactive engagement invitation website with countdown timer, music player, and invitation gallery.

## ✨ Features

### 🎯 Core Features
- **⏰ Countdown Timer** - Live countdown to engagement day (March 12, 2026)
- **🎵 Music Player** - 50+ trending Bollywood romantic songs
- **📱 Mobile Responsive** - Perfect on all devices
- **🎨 Theme Switcher** - Multiple beautiful themes
- **🌸 Petal Animations** - Romantic falling petals effect

### 💎 Interactive Elements
- **🎉 Confetti Effects** - Click countdown for celebration
- **🎵 Romantic Playlist** - Searchable Bollywood songs
- **📸 Invitation Gallery** - View and download invitation cards
- **💕 Couple Section** - Beautiful photo showcase
- **🎯 Engagement Details** - Date, time, venue information

### 🎨 Design Features
- **Glass-morphism** - Modern blur effects
- **Smooth Animations** - Hover effects and transitions
- **Gradient Backgrounds** - Beautiful color combinations
- **Responsive Grid** - Adapts to all screen sizes

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. **Clone or download** the project files
2. **Add your images** to `assets/` folder:
   - `divya-sumit-engagement.jpg` - Main invitation card
   - `couple-image.jpg` - Couple photo
   - Music files in `assets/music/` (optional)
3. **Open `index.html`** in your browser
4. **Enjoy!** 🎉

### Local Development
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```

## 🎨 Customization

### Adding Your Own Cards

1. **Via Upload**: Use the drag & drop area on the website
2. **Manually**: Add new card items to the HTML

### Changing Colors

Edit the CSS variables in `style.css`:

```css
:root {
    --primary-color: #d4a574;    /* Main theme color */
    --secondary-color: #8b5a3c;  /* Dark accent color */
    --accent-color: #f4e4d4;      /* Light background */
}
```

### Music

Replace the audio source in `index.html` with your own music file.
